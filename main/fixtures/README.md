# Фикстуры для базы данных

Этот модуль содержит фикстуры для заполнения базы данных тестовыми данными при запуске приложения.

## Что создается

### Пользователи
- 1 администратор (admin@example.com / admin123)
- 9 обычных пользователей (user@example.com / user123)

### События
- 12 различных событий разных типов:
  - День рождения
  - Конференция
  - Мастер-класс
  - Вечеринка
  - Встреча
  - Тренинг
  - Концерт
  - Фестиваль
  - Экскурсия
  - Семинар
  - Тур
  - Завершенное мероприятие

### Связи
- Пользователи приглашены на события
- Некоторые пользователи присоединились к событиям
- Некоторые пользователи лайкнули события

## Управление загрузкой фикстур

Фикстуры загружаются автоматически при запуске приложения, если не указано иное.

Для отключения загрузки фикстур установите переменную окружения:
```bash
LOAD_FIXTURES=false
```

Или в `.env` файле:
```
LOAD_FIXTURES=false
```

## Идемпотентность

Фикстуры идемпотентны - их можно запускать многократно без создания дубликатов. Система проверяет существование данных перед созданием.
